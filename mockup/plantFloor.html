Tap a tray-quantity button to close the current tray. Each tray close fires one ConsumptionEvent against each BOM component LOT, increments the parent Container's running count, and unlocks the next tray. Container completes when accumulated trays × PartsPerTray reaches the configured capacity.

No per-piece serial · ConsumptionEvent fires per tray.

Per-Tray Close · ByWeight

Configured TargetWeight = 18.4 lb · OmniServer scale → PLC TrayFullFlag

On ByWeight lines, the operator does not enter a count. The OmniServer scale publishes the per-tray weight; the PLC compares against TargetWeight on ContainerConfig and asserts TrayFullFlag at threshold. The MES reads the flag and writes the close. The operator's only action is removing the tray from the scale and starting the next tray.
